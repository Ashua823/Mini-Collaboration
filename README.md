# Mini-Collaboration

## 项目设计
Mini-Collaboration 协作平台

## 项目概述
一个支持多用户在线、实时通信、文件共享的轻量级协作平台。  
使用 Socket 实现网络通信，支持并发用户，涵盖你学过的所有核心知识。

---

## 📋 核心功能模块

### 第一阶段：基础功能

#### 用户系统
- 注册 / 登录（密码加密存储）
- 用户信息持久化到文件
- 用户验证与会话管理

#### 公共消息区
- 发送 / 查看消息（集合管理）
- 消息持久化（IO 流写入）
- 消息列表查询（Stream 流操作）

#### 在线用户管理
- 实时显示在线用户列表
- 用户上线 / 下线事件处理

---

### 第二阶段：高级功能

#### 私信系统
- 点对点消息发送（并发处理）
- 消息队列缓存

#### 文件上传 / 下载
- 文件 IO 操作（字节流）
- 文件列表展示
- 断点续传模拟

#### 网络通信
- Socket TCP 服务器
- 多客户端支持（线程池）
- 消息协议定义

---

### 第三阶段：架构优化

#### 反射 + 动态代理
- 自动服务注册发现
- AOP 日志切面
- 权限校验拦截器

#### 异常处理系统
- 自定义异常体系
- 统一异常处理
